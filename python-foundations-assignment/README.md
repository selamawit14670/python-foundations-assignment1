# Python Foundations Assignment

## Overview
This repository contains solutions for three Python tasks focused on basic programming concepts such as functions, loops, and conditionals.

## Tasks Implemented

### 1. Prime Number Checker
This program checks whether a given number is a prime number. It uses a loop to test divisibility.

### 2. Vowel Counter
This program counts the number of vowels in a given string. It loops through each character and checks if it is a vowel.

### 3. CLI-Based Calculator
This is a simple command-line calculator that performs addition, subtraction, multiplication, and division using functions.

## Concepts Used
- Functions
- Loops
- Conditional Statements
- User Input Handling
